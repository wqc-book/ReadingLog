思维不设限 2021/7/12 ～ 8/12
= 收获：不同领域发生交叉碰撞会产生新的想法，而且还能让想法倍增，从而会有着不同的角度、想法。人往往会沿着最简单的路径产生联想，即根据过去产生联想，联想约束了横向思维，不再下意识对假设进行质疑，而是迅速跳到结论上，然后又针对某个特定情况设置人为障碍。通过学习不同领域的知识，让自己有着看待事物的另一个角度。把问题的假设倒过来思考，会从一个完全不同的角度看待问题。从相同的角度看待问题，往往只会注意到同样的东西，把当前想法应用到别的人和事，从而影响当前的思维，或者设置约束条件，强迫摸索出解决问题的新方法。反省过去做过的事，重新考虑曾经的想法，用新的眼光看待过去
= 误区：人在解决问题的时候总是想到一个方法就去试验，而不是尽可能想出多的想法，然后进行评估、挑选，往往最先想到的都是普通没有创意的。做一件事，要把可能发生的错误考虑进去，然后在不断的失败中摸索总结，而不是同样的错误一犯再犯。受情绪影响，人们总是等到要失去的时候才肯冒险，有条件做出决定的时候却又畏首畏尾。过去做过的事情不会成为衡量今后应当做什么事的标准。即使机会再好，只要承担不了风险，最终都会放弃。从多个视角看待事情才会考虑的更加全面，切勿先入为主，从而权衡利弊。人总是习惯于先选择目的地，再朝着目的地所在的方向前进
